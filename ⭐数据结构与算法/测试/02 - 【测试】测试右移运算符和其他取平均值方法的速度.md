## 对象
- 右移运算符`<<`，即二进制向右移位，相当于除以2的n次方，不进位
- Math.trunc
- /2
## 条件
- `performance.now()`
- 1000次，迭代次数1_000_000次

## 数据

|        | `>>`    | `\2`    | `Math.trunc` |
| ------ | ------- | ------- | ------------ |
| 时间     | 0.268ms | 4.761ms | 4.466ms      |
| 浮点数下时间 | 4.028ms | 4.945ms | \            |
## 结论：整数下`>>`求平均值速度是`/2`的接近18倍！

## 实验代码
```js
    (function () {
      const N = 100;             // 重复次数
      const ITER = 1_000_000;  // 每轮迭代次数

      function bench(name, fn) {
        let total = 0;
        for (let t = 0; t < N; t++) {
          const start = performance.now();
          for (let i = 0; i < ITER; i++) fn(i);
          const end = performance.now();
          total += (end - start);
        }
        console.log(`${name} 平均耗时: ${(total / N).toFixed(3)} ms`);
      }

      console.log('测试开始（注意：>> 会强制 ToInt32）');

      bench('bitshift >> 1 (i >> 1)', i => i >> 1);
      bench('divide by 2 (i / 2)', i => i / 2);
      bench('Math.trunc(i / 2)', i => Math.trunc(i / 2));

      // 浮点数测试
      console.log('浮点数测试：');
      bench('bitshift >>1 on float (i+0.5 >> 1)', i => (i + 0.5) >> 1);
      bench('divide by 2 on float ((i+0.5)/2)', i => (i + 0.5) / 2);
    })();
```
